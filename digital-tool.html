Organize It!

As apartments and homes get smaller and more expensive, people need affordable, customizable spaces. This solution provides a DIY design tool that simplifies closet customization, maximizes space, and fits users' budgets.Organize it, is a tool in the form of set of grasshopper plug-ins, that lets users design custom closet.

InBetween

This project draws the inspiration from, “Browsing the Latent Space: A New Approach to Interactive Design Exploration for Volumetric Generative Systems”. Aim: To create a tool that will help the user to visualize the latent space to create various design iterations, give them a wider iteration set as a part of the initial design iterations.
